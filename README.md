# Quantityzer

AI-приложение для подсчёта объектов на фото с подсветкой рамками.

## Как запустить локально
```bash
python3 -m http.server 8080
```
Откройте http://localhost:8080

## Деплой на GitHub Pages
1. Создайте репозиторий `quantityzer` на GitHub
2. Загрузите все файлы из этой папки
3. Settings → Pages → Deploy from branch → main → Save
4. Сайт: https://ВашеИмя.github.io/quantityzer/

## Установка на телефон
- **Android**: Chrome → меню ⋮ → «Добавить на главный экран»
- **iPhone**: Safari → Поделиться → «На экран Домой»

## API-ключ
При первом запуске нажмите 🔑 и введите ключ с console.anthropic.com
Ключ хранится только на вашем устройстве.
